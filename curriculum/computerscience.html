Computer Science

Build a full functioning only shopping cart application using ReactJS

Starting point

  • Use https://blog.bam.tech/developper-news/get-the-best-of-your-react-app-design-by-using-material-ui-theme to create your first web application with MaterialUI (https://material-ui.com/demos/).
  • Use https://medium.freecodecamp.org/how-to-make-create-react-app-work-with-a-node-backend-api-7c5c48acb1b0 to create an express backend. This is just an example of how to do it, would need to curate something that works more precisely for your students.

Login

Build a login portal that uses facebook or google to authenticate the user.

Items

Create an items endpoint that will serve up a list of 10 distinct items

Searching

Create a search page that will search for the items served up by the API

Cart endpoint

Create a new API endpoint that will take items, quantities, and the user; and save them in the cart

Add items to a cart

Add a button to the search page that will add items & quantities to your cart via the API

View your cart

Add a GET endpoint to retrieve items from your cart, then use that endpoint to view the items saved.

Add a database

Connect your express endpoints to a postgresql database to save/retrieve data

Checkout

Build a page that will check a cart out
